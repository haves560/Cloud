#系統功能

(1) 區長推播訊息:不論何時何地只要有在區域註冊都能收到訊息。
(2) 區民上傳訊息:反映事件給村長或活動文宣推廣。

(3) 與行事曆結合:讓使用者接收到通知就能馬上進行管理與分享。 
(4) QRcode:藉由每次活動生產QRCode紀錄活動資訊。

(5) 急難訊息回報:發生事件區長發送緊急訊息進行人數統計及回報。 
(6) 與Facebook API結合:讓活動訊息給更多民眾知道。
